# 朋友圈营销文案生成功能需求

## 需求背景

在完成视频生成后，额外生成一段可直接发朋友圈的营销文案（带丰富emoji），方便用户一键复制到朋友圈发布。

## 功能描述

### 现有流程
```
1688爬取 → DeepSeek生成种草文案 → Seedance生成视频 → Remotion合成 → 最终视频
```

### 新增流程
```
1688爬取 → DeepSeek生成种草文案 → DeepSeek浓缩朋友圈文案 → Seedance生成视频 → Remotion合成 → 最终视频 + 朋友圈文案
```

## 详细需求

### 1. 后端 - 朋友圈文案生成
- **输入**: 已生成的种草文案（种草文案.txt）
- **调用**: DeepSeek API
- **输出**: 浓缩后的朋友圈营销文案（30-50字），带丰富emoji
- **保存**: 新建 `朋友圈文案.txt` 文件

### 2. 后端 - 返回给前端
- 在任务完成时，将朋友圈文案内容通过API返回给前端

### 3. 前端 - 展示结果
- 在生成结果页面展示：
  - 最终视频预览
  - 朋友圈文案（带复制按钮）
  - 一键复制功能

## 实现方案

### 后端修改
1. **新增函数**: `generate_wechat_moments_copy()`
   - 调用DeepSeek API浓缩文案
   - 添加emoji到文案中
   - 保存为 `朋友圈文案.txt`

2. **修改 `video_pipeline.py`**
   - 在生成种草文案后调用新函数
   - 将文案路径加入render_data

3. **修改 API 响应**
   - 任务完成时返回朋友圈文案内容

### 前端修改
1. 在结果展示区添加朋友圈文案卡片
2. 添加"复制到剪贴板"按钮

## 关键文件

- `video_pipeline.py` - 新增朋友圈文案生成逻辑
- `backend/services/composition_service.py` - 集成新功能
- `backend/main.py` - API响应添加文案字段
- `frontend/src/app/page.tsx` - 展示朋友圈文案

## 验证方式

1. 运行 `./run.sh` 启动服务
2. 创建新任务，填写1688链接
3. 等待任务完成
4. 检查结果页面是否展示朋友圈文案
5. 点击复制按钮测试复制功能
